# Claude Mandates — Public Resume Repository

## Repository Structure

This is the public-facing resume repository. The root `index.html` is the primary entry point for GitHub Pages.

```
index.html                      ← PRIMARY: UpWork / Embedded EE (GitHub Pages root)
assets/                         ← Shared media (KCNSC-logo.jpg, office_1.mp4)
archive/                        ← Superseded versions (read-only)
  2025/                         ← 2025 historical snapshots
  2026/                         ← 2026 historical snapshots
```

## Guidelines

### Primary Resume (index.html)
1. **Technical Primacy:** Lead with technical skills and specific MCU/EDA tool expertise.
2. **UpWork Optimization:** Focus on keywords: "Embedded C", "PCB Layout", "Firmware", "Sensor Fusion", "Edge AI".
3. **Aesthetics:** High-quality HTML/CSS presentation.

### General Rules
1. **No Sensitive Data:** Do not commit personal identifiers beyond what is already published.
2. **GitHub Pages Compatibility:** Static files only. No build step.
3. **HTML/CSS Quality:** Semantic HTML5, responsive CSS, print-friendly `@media print` styling.
4. **Archive Policy:** Files in `archive/` are read-only historical snapshots. Never modify archived content.
