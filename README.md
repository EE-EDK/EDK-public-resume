# Ethan Kunz — Public Resume

Publicly accessible resume repository. Hosted via GitHub Pages.

**Live site:** [https://ee-edk.github.io/EDK-public-resume/](https://ee-edk.github.io/EDK-public-resume/)

## Repository Structure

- `index.html`: Primary resume (Embedded Systems & Electrical Engineering focus).
- `assets/`: Shared media assets (logos, videos).
- `archive/`: Superseded resume versions (read-only).

## Development

This repository is a static site. Changes to `index.html` are automatically reflected on the live site after pushing to `main`.

### Assets
All assets should be placed in the `assets/` directory.

### Archive
Historical snapshots are stored in `archive/` by year. Do not modify archived content.
