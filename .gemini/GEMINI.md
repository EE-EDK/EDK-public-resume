# Gemini Mandates — Public Resume Repository

## Objective

Maintain a publicly accessible resume repository with a primary UpWork/Embedded EE focus (`index.html`).

## Core Mandates

1. **Technical Primacy (index.html):** Lead with technical skills and project outcomes.
2. **Embedded & EE Focus:** Emphasize experience with ESP32, STM32, RP2040, MSP430, PCB Design (Altium, KiCad, LTSpice), High-Voltage & Pulsed Power, Industrial Automation.
3. **Web Standards:** Semantic HTML5, accessible markup, responsive CSS. Static site only.
4. **Print Fidelity:** `@media print` styles must produce clean PDF output.
5. **No Build Tools:** GitHub Pages serves files directly.
6. **Archive is Read-Only:** Files in `archive/` are historical snapshots. Do not modify.

## File Structure

- `index.html` — Primary resume (Embedded EE focus, GitHub Pages root)
- `assets/` — Shared media assets
- `archive/` — Historical versions

## Rules

1. Do not modify content without explicit instruction.
2. Validate HTML — all changes must produce valid HTML5.
3. Preserve formatting — layouts are tuned for screen and print.
4. Asset management — all assets live in `assets/`. Use relative paths.
